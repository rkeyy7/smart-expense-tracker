# 📊 Smart Expense Tracker & Personal Finance Dashboard

Una aplicación web full-stack diseñada para ayudar a las personas a tomar el control de sus finanzas personales. Permite registrar ingresos y gastos diarios, organizarlos por categorías y visualizar los hábitos financieros a través de un dashboard interactivo.

## 🚀 El Problema que Resuelve
Muchas personas carecen de claridad sobre en qué gastan su dinero, su capacidad de ahorro y cómo evolucionan sus finanzas con el tiempo. Esta herramienta elimina las suposiciones y convierte los datos cotidianos en información visual y accionable para tomar mejores decisiones financieras.

## 🛠️ Stack Tecnológico
* **Backend:** Python, Flask
* **Base de Datos:** PostgreSQL, SQLAlchemy (ORM)
* **Frontend:** HTML5, Tailwind CSS (CDN)
* **Visualización de Datos:** JavaScript, Chart.js
